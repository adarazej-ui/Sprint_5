# Проект UI-автотестов для Stellar Burgers (Спринт 5)

Инженер по автоматизации: Анастасия
Стек технологий: Python 3.x, Pytest, Selenium WebDriver (Chrome)

## Описание проекта
В рамках данного проекта реализовано покрытие автотестами космического фастфуда **Stellar Burgers** на учебном стенде. Архитектура построена по паттерну Page Object Model (POM).

## Покрытый функционал
- Успешная регистрация и валидация длины пароля.
- Вход в систему со всех 4 точек авторизации.
- Переходы между разделами Конструктора и Личным кабинетом.
- Успешный выход из аккаунта.

## Инструкция по запуску
1. Установите зависимости: `pip install selenium pytest`
2. Запустите тесты из корня проекта: `python -m pytest -v`# Sprint_5
